## Welcome

해당 프로젝트는 인스타그램의 클론 앱니다.

## 필수 사항

- iOS13.0 +
- Xcode12.4 +

## 인스톨 방법

- 해당 프로젝트는 [XCodeGenerator][xcodegen-url]로 프로젝트 파일을 생성합니다.
- 인스톨은 shell script로 간편하게 할 수 있습니다.

```shell
$ brew bundle --file=./Brewfile
$ make project
```
